# Library Management System (C Project)

## Overview
This project is a console-based Library Management System developed in C. It was built to demonstrate a practical grasp of essential programming concepts, including problem resolution, structured coding, modular architecture, and file management. 

The application simulates a real-world back-office system for librarians to manage book inventory, track issuance status, and persist data across sessions using binary file I/O.

## Features
1. **Add New Book:** Enter a unique ID, title, and author to add a book to the catalog.
2. **Display All Books:** View a formatted table of all books in the library and their current availability status.
3. **Search Book by ID:** Quickly locate a specific book's details using a linear search.
4. **Issue Book:** Mark a book as "Issued" (prevents double-issuing).
5. **Return Book:** Mark an issued book as "Available" again.
6. **Data Persistence:** Save the current library catalog to a binary `.dat` file and automatically load it upon restarting the application.

## Project Structure
The project utilizes a modular architecture divided into three main source files:
* `main.c`: The entry point of the program. Contains the main application loop and the interactive command-line menu.
* `library_functions.c`: The core logic engine. Contains the implementation for all library operations and file I/O.
* `library.h`: The header file containing the `Book` struct definition, constant macros (`MAX_BOOKS`), and function prototypes.
* `library.dat`: A binary file generated by the system to permanently store the array of structs.

## Prerequisites
To compile and run this project, you need a C compiler installed on your system.
* **Windows:** MinGW (Minimalist GNU for Windows) providing the `gcc` compiler.
* **IDE:** Visual Studio Code (or any preferred text editor).

## How to Compile
Open your terminal (ensure it is pointed to the directory containing the source files) and run the following command to link and compile the program:

```On Windows:
gcc main.c library_functions.c -o library_app

```
## How to Run
After successful compilation, start the application by running the generated executable:

```On Windows:

.\library_app.exe

```

## Usage Instructions
Upon launching, the system will attempt to load existing data from library.dat.

Use the number keys 1-6 to navigate the menu options.

When adding a book, ensure the ID is an integer. Titles and authors can contain spaces.

Important: Always select option 6 (Save Data to File) before exiting to ensure your latest changes are permanently stored.

Select option 0 to safely terminate the program.
